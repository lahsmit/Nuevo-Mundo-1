Semana 04

Voy a detallarte todo lo que se llevo a cabo durante la semana 04
Desarrollo de Proyectos con Arduino y Programación en CSS.

Introducción

Durante esta semana, exploramos tres lenguajes de programación, discutimos cómo ChatGPT puede ser una herramienta útil en este proceso, y también descubrimos algunas páginas web que hacen que aprender a programar sea divertido, casi como jugar.

Clase Virtual

En el desarrollo web, existen tres lenguajes fundamentales para crear sitios modernos y funcionales: HTML define la estructura y el contenido de la página web, CSS se encarga del diseño y la presentación, y JavaScript añade interactividad y dinamismo. Una manera efectiva de aprender estos lenguajes es a través de Liveweave, un editor donde puedes practicar de forma sencilla y en tiempo real. Si te encuentras con problemas o se te olvidan los códigos, puedes solicitar ayuda a ChatGPT. Por ejemplo, nuestro profesor nos recomendó hacer la siguiente solicitud: "Hola ChatGPT, necesito que seas un desarrollador web y docente para guiarme en mis primeros pasos en programación web con HTML y CSS. ¿Puedo iniciar una consulta?" De esta manera, ChatGPT te guiará paso a paso y te explicará el porqué de cada acción, facilitando la comprensión del proceso. Además de Liveweave, existen otras páginas recomendadas en clase como FLEXBOX FROGGY y FreeCodeCamp, que son excelentes recursos para aprender y practicar. Durante esta clase, también aprendimos a utilizar Netlify, una plataforma que nos permite alojar y mantener los sitios web que creamos. Para empezar, solo necesitas crear una cuenta en su página web " https://www.netlify.com/ ", luego cargar los archivos de tu proyecto y obtendrás la URL para acceder a tu nuevo sitio web. Para finalizar, el profesor nos brindó asesoría separándonos en grupos e ingresando a salas virtuales para abordar las problemáticas específicas que teníamos. Esta interacción directa facilitó la resolución de dudas y problemas durante la clase.

Clase Personal

La expresión personal evoluciona con el tiempo y se refleja en diversos aspectos, incluso en la bitácora. Por ello, es esencial contar con las herramientas adecuadas para realizar cambios cuando sea necesario. No obstante, simplificar este proceso es clave; una forma práctica de hacerlo es copiar una parte del código que contiene el elemento deseado y pegarlo donde sea necesario. Durante la clase, practicamos diversos cambios y tuvimos la oportunidad de plantear preguntas sobre cualquier detalle relacionado con la bitácora, Tuve algunas dudas y necesité la ayuda del profesor, quien me brindó mucha asistencia con mi bitácora. Además, participamos en un proyecto de clase diseñado para fomentar la empatía en nuestros proyectos. Creamos un mapa mental a partir de una necesidad específica, explorando las posibles razones y consecuencias que podrían haberla originado. El objetivo era idear soluciones utilizando Arduino que fueran aplicables en el mundo físico. En nuestro grupo, discutimos la complejidad de vivir en soledad, especialmente para un adulto mayor. Uno de nuestros compañeros realizó una exposición excelente y explicó brillantemente nuestras ideas. ¡Y lo mejor de todo es que ganamos! Esto nos otorgó un punto adicional para el equipo, ¡y estamos listos para afrontar nuevos desafíos!
