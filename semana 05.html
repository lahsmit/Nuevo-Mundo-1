Voy a detallarte todo lo que se llevó a cabo durante la semana 05
Desarrollo de Soluciones Innovadoras para Desafíos de los Objetivos de Desarrollo Sostenible (ODS) y Noticias Actualizadas.

Introducción

Esta semana es ideal para la automatización. Deja atrás la repetición manual de pasos una y otra vez. Sin embargo, asegúrese de mantener el enfoque en cada tarea, ya que cada paso es fundamental.

Clase Virtual

Sería agotador realizar cada cambio manualmente, por eso es mejor integrar Visual Studio y GitHub para que los cambios se reflejen automáticamente en ambos programas. Para lograrlo, primero necesitas descargar Git, que permite que los repositorios locales sean completamente funcionales y te facilitan trabajar en ellos tanto sin conexión como de forma remota. Puedes descargar Git desde: https://git-scm.com/downloads y seleccionar la versión compatible con tu sistema operativo. Durante la instalación, no olvides elegir la opción para integrarlo con Visual Studio Code. Una vez instalado, tendrás que clonar la URL de GitHub desde la opción "Code-HTTPS" y crear la carpeta donde se almacenará todo el proyecto. Luego, utiliza la opción "Ejecutar-Ejecutar sin Depurar", que ya conocemos. Después, ve a la "Terminal" y ejecuta los comandos: git config --global user.name "[nombre de usuario]"[ENTER] y git config --global user.email "[correo electrónico]"[ENTER], remplazando [nombre de usuario]y [correo electrónico]con tus datos. Sigue estos pasos y tendrás todo sincronizado y listo para editar. Al finalizar la clase, el profesor ofreció asesoramiento y pidió a aquellos que tenían dificultades con el proceso que compartían sus pantallas. Aunque al principio experimenté el mismo error que otros, las correcciones que se hicieron lo solucionaron y ahora todo está listo para empezar .

Clase Presencial

En esta sesión, profundizamos en las propiedades y códigos en Visual Studio. Por ejemplo, ¿sabías que para insertar imágenes necesitas utilizar href="nombre de la carpeta/nombre de la imagen.png"? Sin embargo, esto solo funcionará si la imagen deseada se encuentra dentro de la misma carpeta creada para los archivos inicialmente. Además, es crucial asegurarse de que los nombres estén escritos exactamente iguales para que el programa los reconozca correctamente. Si encuentras dificultades al cargar las imágenes, puedes subirlas directamente a GitHub. Espera hasta que aparezca el punto verde, que indicará que todo se ha cargado correctamente. También discutimos sobre los estilos y su uso en Visual Studio, que se refieren a las reglas de estilo CSS aplicadas a un documento HTML. Con estas reglas, puedes editar y administrar el diseño en bloque directamente en el editor. Desde el archivo CSS principal, puedes cambiar el color de estilos para elementos como h1, h2, entre otros. Como en todas las clases, el profesor nos asignó una actividad. Esta vez, consistía en recopilar la mayor cantidad posible de soluciones tecnológicas basadas en uno o más ODS. En nuestro grupo generamos entre 5 y 7 ideas, que abarcan desde la adopción de animales hasta mejoras en la educación. Estamos considerando algunas de estas ideas para el proyecto final.
